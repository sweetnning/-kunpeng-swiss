1.2 · 鲲鹏与深圳4/24
BRAND FOUNDATION

鲲鹏、鹏城与深圳科技馆的一脉共生

01
文化根脉
鲲鹏意象 × 鹏城底色
《庄子·逍遥游》中"水击三千里，抟扶摇而上者九万里"的鲲鹏意象，是刻在中华文脉中突破边界、志存高远的精神符号。深圳别称"鹏城"，既源自600余年海防历史的大鹏所城，更与这份跨越千年的鲲鹏之志深度契合——这座城市自特区建立以来，便以"鹏程万里"的魄力一路突破成长，成为中国科创的核心高地。
02
产业硬核
从文化符号到鲲鹏算力
这份根植于城市的鲲鹏精神，在数字时代完成了从文化意象到硬核力量的具象落地。以"鲲鹏"命名的自主可控计算产业，正是深圳科创精神的核心载体：深圳作为全国鲲鹏产业示范区，鲲鹏算力是支撑鹏城云脑等国家级重大算力平台的核心底座，是深圳自主科创实力的标杆名片。
03
科普窗口
深圳科技馆串联融合
作为深圳城市级科创地标、全球首家数字文明主题科技馆的深圳科技馆（光明），本身就以展现深圳本土科创根脉、传递城市科创精神为核心定位，80%展项自带深圳科创基因。它既是向公众科普鲲鹏算力这一前沿成果的核心窗口，更是将古老鲲鹏文脉、鹏城城市精神与当代科创成就串联融合的载体。

3.4 · 品牌的边界与红线20/24

品牌的边界与红线

科学真实是我们的最高宪法。
四条叙事铁律
1. 可演绎，不可曲解
可以把"量子纠缠"比喻成"鲲鹏号的超距感应通讯"，但不能说它就是心灵感应
2. 可简化，不可错误
可以简化科学原理，但不能传递错误知识
3. 标明虚构边界
所有故事化内容标注："本故事是基于科学原理的文学想象"
4. 科学准确性拥有一票否决权
遇争议时，以科学准确性为最终裁决标准
审核机制
设立"首席科学家 × 首席品牌官"联合审核委员会。所有对外内容，科学准确性拥有一票否决权。
关键词合规红线（禁用表达）
· 禁用"鲲鹏号是真的宇宙飞船"（混淆虚构与现实）
· 禁用"这是深圳最好的科技馆"（违反广告法，姿态傲慢）
· 禁用"不好好学习就不配当鲲鹏队员"（价值观风险）
· 禁用任何贬低其他科技馆或教育机构的比较性话术
· 禁用"最""第一""唯一"等绝对化用语（违反广告法第九条）
以上全部写入《品牌叙事手册》首页——从第一天起，边界清晰，红线明确。
4.1 · 对标与预期21/24

我们不盲目崇拜，我们只精准学习

每一个对标对象，都对应一个我们必须攻克的战略课题。我们清晰地知道：学他们什么，不学他们什么，以及当下要先成为谁。
三大战略对标

三星堆 · 学 IP 矩阵化打法

学：严肃内容 → 人格化 IP 的转化逻辑。
当下策略：启动鲲鹏科学宠物家族，让四大知识域成为可亲近的伙伴。
边界：不为做 IP 而曲解科学。我们的 IP 是对科学的"翻译"，不是"改编"。

故宫文创 · 学 IP 三阶段路径

学：从复制品(1.0)→衍生品(2.0)→IP 化生态(3.0)。
当下策略：我们处于 1.0 向 2.0 过渡。第一年集中打造 1-2 款爆款衍生品。
边界：故宫有 600 年积淀。不在品牌根基不稳时强行 IP 化。

TeamLab · 学沉浸体验

学：用数字技术创造他处无法复制的沉浸式体验。
当下策略：第一年至少制造一个刷屏级馆内体验瞬间。
边界：我们是科技馆，不是光影秀场。惊叹之后必须留下知识。

两个近期务实对标

国家大剧院文创

学习垂直领域文化品牌起步。像大剧院定位"艺术普及"，我们定位"科学启蒙与传播"。第一波衍生品只服务于"来过馆的孩子和想让孩子接触科学的家长"。

同期新建科技馆

开馆前完成对至少 2 家同期科技馆深度调研：首年游客量曲线、引流策略、家长复购率最高的课程类型。不做闭门造车。

我们从不对标的对象中学到什么
深度研究过迪士尼，结论是：不学。迪士尼的核心是"造梦"。科技馆的核心是"求真"。鲲鹏的 IP 必须长在科学的土壤里。这条线，是我们与娱乐巨头的根本分水岭。
我们的对标哲学：向最强者学思路，向同行者学打法，向娱乐巨头学边界。
不因仰望星空而忘记脚下台阶，不因追逐潮流而模糊立身之本。
第一年，我们只证明一件事：鲲鹏，是一个让孩子真正爱上科学的地方。
